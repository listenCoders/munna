JavaScript Arrays Guide

Arrays are used to store multiple values in a single variable. An array is a special variable that can hold more than one value at a time.

1. Introduction to Arrays

Arrays are used to store multiple values in a single variable. An array is a special variable that can hold more than one value at a time.

2. Creating Arrays

You can create arrays in JavaScript in two ways:

  • Using an array literal:
    • let fruits = ["Apple", "Banana", "Cherry"];
  • Using the Array constructor:
    • let fruits = new Array("Apple", "Banana", "Cherry");

3. Accessing Array Elements

You can access an array element by referring to its index number.

let fruit = fruits[0];  // Apple

JavaScript arrays are zero-indexed, meaning the first element is at index 0.

4. Modifying Array Elements

You can change an element by accessing its index:

fruits[1] = "Blueberry";  // Banana is now Blueberry

5. Array Properties and Methods

  • length: Returns the number of elements in an array. 
    let length = fruits.length;  // 3
  • push(): Adds a new element to the end of an array. 
    fruits.push("Date");  // ["Apple", "Blueberry", "Cherry", "Date"]
  • pop(): Removes the last element from an array. 
    let last = fruits.pop();  // Date
  • shift(): Removes the first element from an array. 
    let first = fruits.shift();  // Apple
  • unshift(): Adds a new element to the beginning of an array. 
    fruits.unshift("Apricot");  // ["Apricot", "Blueberry", "Cherry"]

6. Looping Through Arrays

You can loop through array elements using different loops:

  • for loop: 
    for (let i = 0; i < fruits.length; i++) {{
    console.log(fruits[i]);
}}
  • for...of loop: 
    for (let fruit of fruits) {{
    console.log(fruit);
}}
  • forEach() method: 
    fruits.forEach(function(fruit) {{
    console.log(fruit);
}});

7. Common Array Methods

  • concat(): Joins two or more arrays. 
    let moreFruits = ["Elderberry", "Fig"];
let allFruits = fruits.concat(moreFruits);  // ["Apricot", "Blueberry", "Cherry", "Elderberry", "Fig"]
  • slice(): Selects a part of an array and returns a new array. 
    let citrus = fruits.slice(1, 3);  // ["Blueberry", "Cherry"]
  • splice(): Adds/removes items to/from an array. 
    fruits.splice(2, 0, "Blackberry");  // ["Apricot", "Blueberry", "Blackberry", "Cherry"]
  • indexOf(): Searches the array for an element and returns its position. 
    let index = fruits.indexOf("Blueberry");  // 1
  • find(): Returns the value of the first element that passes a test. 
    let result = fruits.find(fruit => fruit.startsWith("B"));  // Blueberry
  • filter(): Creates a new array with every element that passes a test. 
    let result = fruits.filter(fruit => fruit.startsWith("B"));  // ["Blueberry", "Blackberry"]

8. Advanced Topics

  • Multidimensional Arrays: Arrays containing arrays. 
    let matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];
  • Array Methods like map(), reduce(): Perform transformations and reductions on arrays. 
    let lengths = fruits.map(fruit => fruit.length);  // [7, 9, 9, 6]
            
let totalLength = lengths.reduce((total, length) => total + length, 0);  // 31
